A Single Source

This book describes a fully–automated method of electronic edition.

Taking a real–world case, I explain what is and what implies working with digital texts.

Download

Unfortunately, A Single Source isn’t available yet in English. Only its Spanish version, Un solo origen has been released.

It may take some time, because there are some issues which should be fixed first.

Purpose

My aim with this work is to show how a professional method of electronic editing could be achieved following this basic principle:

One source to generate them all.

This document generation from a single source may be achieved using pandoc. But I’m afraid that pandoc has some issues that discourage its adoption by a wider audience.

These are the issues that should be fixed first, before learning pandoc would make sense.
